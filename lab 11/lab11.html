Tema 11.1

Se considera documentul html ex1.html si fisierul persoane.json.
Implementati o aplicatie server folosind Node.js si Express astfel incat la submit pe formularul dat (cu metoda get/post), pentru cheia oferita sa se afiseze ca lista ordonata,
in interiorul paginii cu formularul, in iframe, toate numele complete asociate prenumelui/numelui (conform select) respectiv.

Tema 11.2

Se considera documentul html ex2.html.
Implementati o aplicatie server folosind Node.js si Express astfel incat la submit pe formularul dat, sa se trimita doi identificatori posibili din vectorul "valori" de pe server (prin inputurile de tip text) si tipul de operatie prin unul din butoanele radio.
Serverul va returna rezultatul obtinut, aplicand operatia ceruta pe valorile identificatorilor (de ex. 10 + 10 = 20). Afisarea se va face in locul paginii cu formularul.
Operatiile se vor salva intr-un fisier json sub forma unui vector de obiecte de forma: {'id_1':val1, 'id_2':val2,'operatia':rezultat}(de ex. {'a':14,'b':10,'adunare':24}.

Tema 11.3

Un document HTML contine doua elemente input de tip text avand etichetele "Email" si "Subiect", un element de tip textarea cu eticheta "Mesaj" si un buton cu textul "Trimite mail".
Implementati o aplicatie server folosind Node.js si Express astfel incat la submiterea formei cu metoda POST sa se trimita un mail la adresa introdusa in campul "Email" si avand ca subiect
textul introdus in campul "Subiect" iar continutul mesajului sa fie textul introdus in campul "Mesaj" (folositi modulul nodemailer).
